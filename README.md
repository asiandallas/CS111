# Spotify Statistics 
CS 111 Final Project - First CS project (in C++)!

Source: https://www.last.fm/user/kyrstnhall/library/tracks?from=2021-01-01&to=2021-04-30

The goal of this project was to retrieve my Spotify listening statistics. The program is able to read from a file, display a table, and useful commands such as searching, deleting, most played songs per month, etc.
